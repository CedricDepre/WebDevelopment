Elke sectie is gebouwd uit en h3 tag met speciale CSS zodat het op een knop likt. Verder wordt er voor de korte tekst een span gebruikt. Voor de lange tekst in de accordeon wordt een div gebruikt

Als je op een gesloten accordeon drukt worden er meerder attributen gewijziogd in de DOM waardoor onder andere CSS wordt veranderd.

Wanneer je een accordeon wil maken hoef je enkel in je broncode meegeven dat je een accordeon wil gebruiken en welke elementen deel moeten maken van de accordeon. Voor de rest zorgt JQuery dat alle elementen de juist attributen met attribuutwaarden aanvult.
